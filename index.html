The Guessing Game

Game Logic

Here is how the game operates:

  1. A winning number between 1-100 will be randomly generated.
  2. Object of the game is to guess the randomly generated number correctly.
  3. Input your guess into the textbox below and press submit to submit your answer.
  4. If you guess the winning number, you win! Otherwise, you can try again.
  5. If you cannot guess within 5 tries, the game is over and you lose.
  6. You may restart the game by pressing "Reset."
